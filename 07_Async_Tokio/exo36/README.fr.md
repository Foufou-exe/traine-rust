# Exercice 36 : Gestion d'Erreurs Asynchrone

## Objectif
Écrire du code robuste en gérant correctement les erreurs avec l'opérateur `?` dans un contexte asynchrone.

## Concepts abordés
- Retourner `Result` depuis `main`.
- L'opérateur de propagation d'erreur `?`.
- Typage dynamique des erreurs (`Box<dyn std::error::Error>`).
