# Exercice 30 : Valeurs de retour Asynchrones

## Objectif
Apprendre à définir des fonctions asynchrones qui retournent des données et récupérer ces données dans le programme principal.

## Concepts abordés
- Définition de fonctions `async fn`.
- Le type `Future` (implice).
- Récupération du résultat avec `.await`.
