# Exercice 34 : Canaux de Communication (MPSC)

## Objectif
Faire communiquer des tâches asynchrones entre elles par envoi de messages, plutôt que par partage de mémoire.

## Concepts abordés
- Pattern **Producer-Consumer**.
- `mpsc` (Multi-Producer, Single-Consumer).
- Envoi (`send`) et réception (`recv`) asynchrones.
