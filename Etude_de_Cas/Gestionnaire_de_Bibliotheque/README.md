## Objective
Apply basic Rust concepts in a mini-project: managing a collection of books.

## Concepts Covered
- **Structs**: Organizing data (Book, Library).
- **Ownership & Borrowing**: Passing book references without losing them.
- **Option & Result**: Handling cases where a book is not found or unavailable.
- **Implementation (impl)**: Encapsulating logic inside methods.
