# Étude de Cas : Gestionnaire de Bibliothèque

[![English](https://img.shields.io/badge/lang-en-red.svg)](./README.md)

## Objectif
Appliquer les concepts fondamentaux de Rust dans un mini-projet : la gestion d'une collection de livres.

## Concepts Abordés
- **Structs** : Organisation des données (Livre, Bibliothèque).
- **Ownership & Borrowing** : Passage de références de livres sans en perdre la possession.
- **Option & Result** : Gestion des cas où un livre est introuvable ou déjà emprunté.
- **Implémentation (impl)** : Encapsulation de la logique dans des méthodes.
